# Weather Forecast App 🌦️

אפליקציה מודרנית ומעוצבת לחיזוי מזג אוויר עבור היום ומחר. המשתמש מזין מיקום ומקבל תחזית מפורטת, נוחה לשימוש ומעוצבת ברמה הגבוהה ביותר.

## 🚀 תכונות עיקריות (Features)
- **חיפוש מיקום חכם**: השלמה אוטומטית (Autocomplete) בזמן הקלדה מבוססת קואורדינטות גיאוגרפיות.
- **תחזית ליומיים הקרובים**:
  - **היום**: טמפרטורה נוכחית, מינימום/מקסימום, לחות, מהירות רוח, ותחזית שעתית.
  - **מחר**: פירוט של תנאי מזג האוויר הצפויים מחר.
- **עיצוב דינמי (Dynamic & Premium UI)**: 
  - רקע משתנה בהתאם למזג האוויר במיקום שנבחר (שמשי, גשום, מעונן, מושלג וכו').
  - אפקט זכוכית (Glassmorphism) מודרני ונקי.
  - תמיכה מלאה במובייל ובדסקטופ (Responsive Design).
  - מעברים ואנימציות חלקות (Micro-animations).
- **ללא צורך במפתח API**: שימוש ב-API החינמי והפתוח של **Open-Meteo** (הן לחיפוש והן למזג האוויר), מה שמאפשר הרצה מיידית ללא הגדרות מסובכות.

## 🛠️ טכנולוגיות (Tech Stack)
- **HTML5**: מבנה סמנטי ונגיש.
- **CSS3 (Vanilla)**: עיצוב מותאם אישית (Custom Properties), גריד (Grid), פלקסבוקס (Flexbox), ואפקטים מתקדמים.
- **JavaScript (ES6+)**: ניהול הלוגיקה, פניות ל-APIs, ועדכון דינמי של ה-DOM.
- **Lucide Icons**: אייקונים מודרניים ונקיים לעיצוב פרימיום.
- **Google Fonts (Outfit)**: פונטים מודרניים למראה נקי.

## 📁 מבנה הפרויקט (Project Structure)
```text
├── index.html       # המבנה של האפליקציה (HTML)
├── style.css        # עיצוב, אנימציות ומערכת הצבעים (CSS)
├── app.js           # לוגיקה, חיפוש, קריאות API ודינמיות (JS)
└── README.md        # תיעוד הפרויקט
```

## 💻 הרצה מקומית (Local Development)
כיוון שהפרויקט מבוסס על HTML/CSS/JS נקיים, ניתן להריץ אותו בשתי דרכים פשוטות:
1. לפתוח את הקובץ `index.html` ישירות בדפדפן.
2. להשתמש בשרת מקומי פשוט (למשל VS Code Live Server או `npx serve .`) לחוויית פיתוח אופטימלית.
